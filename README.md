**Question:**  
Что такое MMU в Cortex-A?

**Short answer:**  
MMU переводит виртуальные адреса в физические и обеспечивает защиту памяти.

**Long answer:**  
В Cortex-A MMU нужен для трансляции адресов, разграничения доступа к памяти и поддержки виртуальной памяти. Это важно для Linux и других систем с многозадачностью.
